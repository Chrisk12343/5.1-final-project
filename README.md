# CSC102 Final Project

A web-based educational project by Christopher Krawczyk featuring interactive games, form validation, and various JavaScript exercises.

## Project Overview

This project is a collection of web-based interactive applications and games built with HTML, CSS, and JavaScript. It serves as a learning portfolio demonstrating fundamental web development skills.

## Features

### Core Pages

- **Home Page** (`index.html`) - Welcome page with navigation to all project sections
- **Navigation Hub** (`nav.html`) - Central hub to access all project features

### Interactive Applications

1. **Rock, Paper, Scissors Game** (`game.html`)
   - Two-player game interface
   - Visual player selection display
   - Automated game logic and result announcement

2. **Login/Form Validation** (`Login.html`)
   - User input validation system
   - First name and last name input fields
   - Zip code numeric validation
   - Real-time input status feedback
   - Background audio element

3. **Cat Table** (`cat table.html`)
   - Styled data table with cat-related content
   - Professional table formatting

4. **String Manipulation** (`strings.html`)
   - String processing exercises
   - Function demonstrations

5. **Image Movement** (`move image.html`)
   - Interactive image positioning
   - Dynamic element manipulation

6. **Game Boards** (`boards.html`)
   - Additional game or board interface
